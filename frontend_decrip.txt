MDR Web Helpcentre — Frontend UI Breakdown
1. Overall Product Purpose
This is an AI-powered Helpcentre and Knowledge Retrieval interface for the MDR Web TB platform.
Primary purpose:


Allow users to search TB-related information


Retrieve synthesised responses from a RAG-based chatbot


Display referenced sources and extracted chunks


Provide quick access to onboarding/help resources


Surface contextual FAQs


This is NOT a chat interface.
This is a search-first knowledge exploration interface.

2. High-Level Layout Structure
The interface can be divided into 5 major sections:
----------------------------------------------------HEADER / NAVBAR----------------------------------------------------HERO SEARCH SECTION----------------------------------------------------LEFT CONTENT AREA        | RIGHT RESPONSE PANEL                         |- Quick Guides           | - Synthesised LLM Answer- FAQs                   | - Referenced Sources                         | - Source Chunks                         | - Feedback Actions----------------------------------------------------
The page uses a:

2. Trustworthy answers

3. Low cognitive load

5. Performance
